# ⏳ Age-Calculator

A small utility to calculate a person's age from a date of birth. Provides precise age in years, months and days, accounting for leap years and calendar boundaries.


## 🌐 Live Demo
Check the live demo of Age-Calculator here:  
## [Live Demo Link](https://GemachisTesfaye.github.io/Age-Calculator/)  


## 🚀 Features
- ✅ Calculate age from a birth date to today or to a target date
- 🧮 Precise output in years, months and days
- ♻️ Handles leap years and month-boundaries correctly
- 🔍 Input validation and helpful error messages
- ⚡ Lightweight and easy to integrate (web or CLI)
- 🔁 Supports Date objects and ISO date strings
- 🔧 Small API surface for easy testing and embedding

## 👀 What you'll see
- Clear human-readable result: "X years, Y months, Z days"
- Errors for invalid formats or impossible dates
- Edge-case handling for Feb 29, month ends and same-day birthdays


## 🧪 Tests & Edge Cases
- Add unit tests for:
    - Leap-year birthdays (Feb 29)
    - End-of-month boundaries (Jan 31 -> Feb)
    - Invalid input formats and future DOBs

## ❓ FAQ
- Q: What formats are accepted?  
  A: ISO date strings (YYYY-MM-DD) and JS Date objects.
- Q: Does it account for timezones?  
  A: Use date-only values or normalize to UTC for deterministic results.
